Crit Night, a project in Django

Crit Night is designed to accompany an informal art discussion event held at a Kansas City venue. This Django application incorporates: Moderator and Artist accounts, Logins with validation, participants may create, read, update and delete their own artwork entries, and a star-rating voting system with comment fields for contributing to the artistic dialog.

Development Internship, Trivial, Inc.

During the Development Internship with Trivial I became acclimated to the Agile environment as part of a professional development team. This was my first remote startup role. I contributed to a JavaScript action library to integrate actions into the codebase of a RESTful application. I used the Postman API platform and Mocha for testing within our automation product.
